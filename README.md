# Sistema de Gestión de Reciclaje

## Descripción
Este proyecto es una aplicación desarrollada para gestionar el proceso de reciclaje, permitiendo registrar usuarios, programar rutas de recolección y generar reportes sobre los materiales reciclables recolectados.

## Objetivos
- Facilitar el registro de usuarios.
- Gestionar rutas de recolección.
- Registrar materiales reciclables.
- Generar reportes de reciclaje.

## Tecnologías utilizadas
- Python
- HTML
- CSS
- SQL
- GitHub


## Instalación

1. Clonar el repositorio.
2. descomprimir la carpeta
3. Abrir la carpeta del proyecto.
4. Ejecutar:

```bash
index.html
```

## Cómo visualizar los prototipos

Abrir los archivos HTML ubicados en la carpeta `templates`.
